Purple Hills Tools

Here is a collection of useful pages for examining the web. Some of these tools are useful for developing your web pages. Also included are some tools for XML and web services debugging. Please Enjoy!

Page Source Viewer
Enter a URL and see exactly what the server sent. There is no interpretation of JavaScript, there is no attempt to parse XML (and fail), there is no processing of redirect commands.
UTF-8 Genie
Enter text and see the unicode and UTF-8 equivalents.
Sudoku Solver
Fustrated by a particular SUDOKU puzzle? Enter the starting values here and have the puzzle solved. You can also play and make your own sudoku puzzle and see IF it can be solved.
Page Cleaner
Useful on a slow link, to avoid loading graphics and other large files attached. Leaves the textual content formatted in a simple page that can be easily read.
URL Hacker
Paste a URL with a bunch of parameters and submit. This will make a form with all the parameters, and allow you to make GET requests with different URL parameters.
LaTeX
enter text and see the LaTeX equivalents.
Threads
produces a thread dump to show what tomcat is doing.
Decimal Formatter
Experiment with decimal formatting in Java.
HTTPRequest Helper
Demonstrating the various parts of the HTTP request path parsing.
XML Grinder
Load XML in either by reading from a web URL, or by pasting from your text editor. This will parse the XML. It will allow you to clean up, reformat, and validate the XML using S-XML (Simple XML) rules.
